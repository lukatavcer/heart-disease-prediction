# Heart disease prediction
## Algorithms used
1) Decision trees
2) Random forest
3) SVM

## Data
[Cleveland heart disease data set 2016](https://archive.ics.uci.edu/ml/datasets/heart+disease)

Correlations using Pearson's coefficient
